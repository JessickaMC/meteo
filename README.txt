Vous trouverez dans ce dossier les fichiers nécessaires au bon fonctionnement du projet météo.
Vous y trouverez les fichiers suivants:
- index.html
- meteo.js
- config.json
- style.css
- README.txt

Ce projet nécessite l'installation de Nodes.js.
Pour se faire, allez sur le site officiel de Nodes, téléchargez et installer Nodes.js.
Enfin, depuis le terminal installer le module http-server avec la commande suivante : npm install http-server

Pour lancer l'application, il vous suffit d'ouvrir le terminal, placez-vous dans le dossier contenant le projet
et tapez la commande suivante : http-server

Si l'application ne fonctionne pas, il est possible que le problème soit dû aux cookies du navigateur.
Dans ce cas, allez dans les paramètres du navigateur et supprimer les cookies et caches.
